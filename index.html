Command & Conquer Linux Port

Open-source GPL port running natively in your browser via WebAssembly

Red Alert
WASM
Requires original game data
Tiberian Dawn
WASM
Requires original game data
You need your own game data. Click a game above and use the "Open Game Folder" button to select your Red Alert or Tiberian Dawn data directory (the folder containing REDALERT.MIX / CONQUER.MIX). Game assets are loaded locally — nothing is uploaded.

Browser: Chrome or Edge required (SharedArrayBuffer + File System Access API). Firefox with dom.fs.enabled may also work.

Source: github.com/hughobrien/battlecontrol (GPL v3, based on the EA open-source release)
